# TxtQR - 文本转二维码桌面应用

一个基于 Rust 和 eframe/egui 开发的桌面应用程序，支持将文本转换为二维码，并提供分页显示和二维码识别功能。

## ✨ 功能特性

- **文本转二维码**：将任意文本（包括中文）转换为二维码
- **智能分页**：自动将长文本分割为多个二维码页面
- **二维码识别**：从剪贴板中的图片识别二维码并转换为文本
- **中文支持**：完美支持中文字符显示和编码
- **可配置参数**：
  - 每个二维码的最大字符数（50-2000字符）
  - 二维码显示尺寸（128-1000像素）
- **用户友好界面**：简洁直观的图形界面

## 🚀 安装和运行

### 环境要求

- Rust 1.70+
- 支持的操作系统：Windows, macOS, Linux

### 安装步骤

1. 克隆项目：
```bash
git clone <repository-url>
cd txtqr
```

2. 安装依赖并运行：
```bash
cargo run --release
```

## 📖 使用说明

### 生成二维码

1. 在文本输入框中输入要转换的文本
2. 调整"每个二维码字符上限"和"显示尺寸"参数
3. 点击"生成二维码并分页"按钮
4. 使用"上一页"/"下一页"按钮浏览不同页面的二维码

### 识别二维码

1. 将包含二维码的图片复制到剪贴板
2. 点击"从剪贴板识别二维码"按钮
3. 识别结果将自动填入文本输入框

### 参数说明

- **字符上限(chunk)**：每个二维码可容纳的最大字符数，建议100-1200字符
- **显示尺寸**：二维码图片的像素尺寸，128-1000像素

## 🛠️ 技术栈

- **GUI框架**：eframe + egui
- **二维码生成**：qrcode (Rust)
- **二维码识别**：rqrr
- **剪贴板操作**：arboard
- **图片处理**：image

## 📦 依赖项

```toml
[dependencies]
eframe = "0.27"
egui = "0.27"
qrcode = "0.13"
image = "0.24"
rqrr = "0.6"
arboard = "3.3"
```

## 🔧 构建

### 开发模式
```bash
cargo build
```

### 发布模式
```bash
cargo build --release
```

## 📋 注意事项

- 单个二维码的容量受版本和纠错级别限制
- 中文字符会占用更多空间，建议适当调整字符上限
- 确保系统有中文字体支持以获得最佳显示效果
